🚀 如何使用
1. 啟動程式（執行 .exe）
2. 在主畫面中：
   選擇交易日期
   按下「下載」按鈕
3. 等待資料下載完成(約5分鐘)
4. 可於介面檢視或到輸出資料夾查看下載結果

若資料來源暫時無法連線，程式會出現錯誤訊息，請稍後再試。

📡 資料來源
⚠ 各券商提供之網頁資料（係來自TW證券交易所 / 櫃買中心 / 開放資料平台）。
來源：台灣證券交易所每日資料下載(為部份資料，並不全面)
欄位：分點代號、買進金額、賣出金額、成交比重等
更新頻率：依交易所公告
使用限制：請遵守來源網站的授權與使用條件

📝 注意事項
本工具僅提供資料下載功能，不構成任何投資建議
請遵守資料來源的使用規範
若大量下載資料，請注意 API 流量限制
